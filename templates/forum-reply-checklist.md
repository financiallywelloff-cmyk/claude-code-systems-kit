# Forum Reply Checklist

Use this before replying in a community thread.

1. Read the original post and understand the actual ask.
2. Read existing comments.
3. Ask: how can I deliver value here?
4. Do not repeat what someone already said.
5. Add one missing thing: a caution, next step, sharper framing, tradeoff, example, or diagnostic.
6. Keep it 2-5 sentences.
7. Link only if the link directly helps.

## Draft Shape

```text
[Acknowledge the useful part or actual blocker.]
[Add the missing caution/next step/tradeoff.]
[Optional link embedded naturally, usually on "here."]
```
