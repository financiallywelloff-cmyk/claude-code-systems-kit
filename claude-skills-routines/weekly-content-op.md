# Weekly Content Op

This is the shape I use when one long-form idea needs to become useful distribution assets.

## Inputs

- One long-form video, transcript, build log, or recorded walkthrough.
- Search Console query/page export.
- Existing site pages that already have impressions.
- One business priority for the week.

## Output

- One owned asset: blog, workflow page, tool, or repo.
- One external proof asset: community post, DEV/Hashnode post, GitHub README, or directory submission.
- One short-form/social batch.
- One measurement note.

## Rule

Do not turn every video into SEO.

If the video has a workflow, make a workflow page or tool.

If the video is mostly story, use it for social/newsletter.

If Search Console is already testing a related page, refresh that page first.
